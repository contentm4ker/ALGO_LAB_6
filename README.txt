Сборка:
-> Зайти в каталог для сборки build/
-> Написать команду "cmake .."
-> Написать команду "make"

Запуск программы:
-> Чтобы запустить программу, которая ищет все вхождения шаблона в текст
необходимо написать полный путь до исполняемого файла, который находится
в каталоге для сборке build/ (kmp). Далее необходимо ввести две строки
подряд, где первая строка является шаблоном, а вторая - текстом.
-> Чтобы запустить программу, которая определяет, является ли первая
введенная строка циклическим сдвигом второй необходимо написать полный
путь до исполняемого файла, который находитсяв каталоге для сборке build/
(shift).

Запуск тестов:
-> Необходимо зайти в каталог сборки, найти и зайти в каталог test/,
где после сборки будет находится исполняемый файл с именем test,
который необходимо запустить, указав полный путь до него.
