Лабораторная работа №5

Сборка:
-> Создать каталог для сборки build
-> Зайти в каталог для сборки build/
-> Написать команду "cmake .."
-> Написать команду "make"

Запуск программы:
-> Чтобы запустить программу, которая ищет все вхождения множества шаблонов в текст
необходимо написать полный путь до исполняемого файла, который находится
в каталоге для сборки build/src (ps).
-> Чтобы запустить программу, которая ищет все вхождения шаблона с масками в текст
необходимо написать полный путь до исполняемого файла, который находится
в каталоге для сборки build/src (joker).

Запуск тестов:
-> Тесты автоматически запускаются во время сборки. В ситуации, когда тесты
не проходят, исполняемые файлы ps и joker не создаются.
-> Необходимо зайти в каталог сборки, найти и зайти в каталог test/,
где после сборки будет находится исполняемый файл с именем test1 и test2,
которые необходимо запустить, указав полный путь до них.
